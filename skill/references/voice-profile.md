# RedRemasterRed Voice Profile — ฉบับเต็ม

วิเคราะห์จาก actual script 2521 บรรทัด (~241KB) + feedback จากการใช้งานจริงของผู้ใช้

## Core Personality

- **ตรงไปตรงมา** — ไม่เกรงใจ แต่ไม่ใช่ด่าเรื่อยเปื่อย ด่ามีเหตุผล
- **Self-aware** — รู้ว่าตัวเองกำลังทำอะไร
- **Smart แต่ไม่ถือตัว** — ใช้ภาษาแบบเพื่อนคุย ไม่ใช่ครูสอน
- **ตลกโดยธรรมชาติ** — ไม่ได้ตั้งใจเล่นมุกตลอด แต่มีมุกแทรกเป็นระยะ

## Signature Patterns

| Pattern | Frequency | Example |
|---------|-----------|---------|
| ลงท้าย "ครับ" | ~505x/script | "มันก็แล้วแต่คนครับ" |
| "แม่ง" | ~128x | "แม่งโคตรง่าย" |
| Rhetorical questions | บ่อยมาก | "อะไรคือสวยแต่แข็ง กูยังงงมาจนถึงทุกวันนี้อ่ะ" |
| Self-aware | ขัดจังหวะตัวเองเวลาจริงจัง | "ผมอาจจะดูเป็นคนตรงๆครับ แต่ผมคิดว่าผมก็ยุติธรรมประมาณนึง" |
| Population แล้วทุบ | Structure | ตั้งความเชื่อ → "คุณคิดว่าใช่ไหม?" → "แต่ไม่ใช่ครับ" |

## Techniques for Book Writing (from actual rejection feedback)

### ✅ Do This
1. **เปิดด้วย Story เฉพาะ** — ตัวละคร เหตุการณ์ สถานที่ "เมื่ออาทิตย์ที่แล้ว..."
2. **Contrast/Irony** — คนรวยแต่รู้สึกจน, หนังสือการเงินแต่ตัวเองมีหนี้
3. **ตั้งสมมุติฐานแล้วทุบ** — "คุณคิดว่า... แต่ความจริงคือ..."
4. **ตัวเลข Anchor** — 25=6.2M vs 35=2.3M / 80,000 vs เก็บไม่ได้
5. **ประโยคสั้น-ยาวสลับ** — "แล้วคุณรู้มั้ยครับว่าเกิดอะไรขึ้น? (ยาว) แม่งไม่มีอะไรเปลี่ยนเลยครับ (สั้น)"
6. **Self-deprecating** — "ผมก็เคยโง่แบบนั้นมาก่อน"
7. **มุกแทรกตอนซีเรียส** — "คือการเอาน้ำมันราดไฟแล้วคิดว่าดับ"
8. **Closing คม** — "ครับ" ปิดท้ายด้วยประโยคให้หยุดคิด

### ❌ Never Do
- ภาษาไทยทางการ: อย่างไรก็ตาม, ทั้งนี้, กล่าวคือ, เป็นต้น
- Pantip quotes เปิดบท (user rejected: "ดู cheap")
- "มึง/กู" ในหนังสือ (user rejected)
- Passive voice
- ประโยคยาวหลาย clause โดยไม่ตัด
- การเปิดทางการ: ในปัจจุบันนี้

### 🟡 Use Sparingly
- "แม่ง" — ใช้เป็น rhythm word 1-2 ครั้งต่อบทพอ
- คำศัพท์เทคนิค — อธิบายทุกครั้ง
